Introduction

Nmap is a free network discovery tool located on your kali linux machine. A full manual can be found here: nmap. You might need to reference this if your scans are too obtrusive or for instance: scans show all the hosts as off line. You might need to add somethign like the --disable-arp-ping argument.

Arguments

Here's the a small list of available options you can use to modify your nmap scripts

Argument Evasive Description
--disable-arp-ping True Use this if you need to disable any arp ping requests.
-oN/-oX/-oG <file> Null Parameter used to save to a file. I prefer -oN.
--version-intensity Null Evasivness depends on the level of intensity 0-9.
